# MediaSamples


#### 主要内容

`Android` 音视频学习案例，你将从这个 `Samples` 中学到如下内容：

- [x] `MediaCodec` 同步、异步处理方式
- [x] `MediaCodec` 编码 `Camera2` 数据录制 `MP4`
- [x] `MediaMuxer` 的使用
- [x] `AudioRecord` 采集音频数据，
- [x] 音频数据与视频数据通过 `MediaMuxer` 合成 `MP4`
- [x] OpenGL ES视频渲染及画面矫正
- [ ] `Camera` 帧数据处理，后续补充完善
- [ ] 继续完善和补充
